Next Steps
  1. Open demimond_ardenverse_site.html in a text editor.
  2. Find the lines near the top of the <script> block:
    const ADMIN_HASH = 'PASTE_HASH_HERE';
    const ADMIN_SALT = 'PASTE_SALT_HERE';
  3. Replace PASTE_HASH_HERE with the ADMIN_HASH value above.
  4. Replace PASTE_SALT_HERE with the ADMIN_SALT value above.
  5. Save the file. To log in, open the browser console on your site and run adminLogin().
  6. To change your password later, just run this tool again and replace both values.
