Generate Password Hash
Enter your chosen admin password. This tool derives a PBKDF2 hash using a random salt — the output is what gets pasted into your site. Your actual password is never stored or transmitted anywhere.
Output — paste both values into js/auth.js
ADMIN_HASH
ADMIN_SALT
Next Steps
  1. Open js/auth.js in a text editor.
  2. Find the two constants near the top of the file:
    const ADMIN_HASH = '...';
    const ADMIN_SALT = '...';
  3. Replace the existing values with the ADMIN_HASH and ADMIN_SALT shown above.
  4. Save the file. To log in, open the browser console on your site and run adminLogin().
  5. To change your password, run this tool again and update both values.
Keep this file private. Do not share it or host it publicly — it is a local tool only. Close this tab when you are done. The hash is one-way, but there is no need to leave the generator accessible.
